## We believe that writing is about content, about what you want to say - not about fancy formatting.

# Markdown

### Syntax

# '#' This is H1
# '##' This is H2
# '###' This is H3

and so on

### List

1. '1.' + Space;
*  '*'  + Space;
-  '-'  + Space;

### Quote

> '>' + text;

### Pictures and Links

Picture: '![](){ImgCap}{/ImgCap}'

Link: []()

### Emphasize and Strong

*emphasize* '*text*' or '_text_'

**strong** '**text**'
